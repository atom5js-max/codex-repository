# 시운전 기록

## 시운전-001 | 권취기 장력제어 시스템 (2024-02)

**현장**: A공장 슬리팅 라인  
**구성**: LS S300 인버터 × 2 + HBM 로드셀 + LS XGB PLC + LabVIEW 상위 PC

### 인버터 파라미터 설정 기록 (S300)

| 파라미터 | 설명 | 설정값 |
|---------|------|--------|
| DRV-05 | Frequency Reference | 3 (RS-485 Communication) |
| DRV-06 | Run/Stop Command | 3 (RS-485 Communication) |
| COM-01 | Station No | 1 (권출), 2 (권취) |
| COM-02 | Baudrate | 9600 bps |
| COM-03 | Parity | None |
| COM-04 | Stop Bit | 1 |
| CON-01 | Control Mode | 1 (Speed Control) |
| BAS-01 | Max Frequency | 60.00 Hz |
| BAS-09 | Accel Time | 5.0 s |
| BAS-10 | Decel Time | 5.0 s |

### Modbus 레지스터 맵 (S300)

| 기능 | 레지스터 | 비고 |
|------|---------|------|
| 운전지령 | 0x0001 | 0=Stop, 1=Run Fwd, 2=Run Rev |
| 주파수지령 | 0x0002 | 단위: 0.01 Hz (예: 3000 = 30.00 Hz) |
| 출력 주파수 | 0x000D | Read Only |
| 출력 전류 | 0x000E | Read Only, 단위: 0.1A |
| 인버터 상태 | 0x000C | 비트 확인 필요 |

### 로드셀 조정 기록

- 로드셀 용량: 100 kgf  
- 증폭기 출력: 4-20 mA  
- PLC AI 모듈: 4-20 mA, 분해능 16000 count  
- 무부하 출력: 4.01 mA (count: ~26)  
- 만부하 출력: 20.00 mA (count: 16000)  
- 스케일링: 장력값(kgf) = count × 100 / 16000  
- 필터 상수: 50 ms (기계 진동 상태 따라 조정)

### LabVIEW 데이터 수집 설정

- 수집 주기: 100 ms  
- DB Insert 주기: 500 ms (5개 데이터 묶음)  
- DB: SQL Server 2019 LocalDB  
- 연결 방식: ADO.NET, Prepared Statement 사용  
- Queue 최대 크기: 1000

## 시운전-002 | 압연기 Modbus 다축 제어 (2024-05)

**현장**: B공장 냉간 압연 라인  
**구성**: LS XGB PLC + LS S300 인버터 × 4대 (Modbus RTU)

### 통신 배선 규칙

- 통신 케이블: RS-485 쉬엘드 twisted pair, 0.5 sq 이상  
- 연결 방식: 데이지체인(Daisy Chain), T-분기 금지  
- 종단저항: 버스 양 끝단에만 120 Ω 설치  
- 최대 노드 수: 32 (RS-485 표준)

### 국번 할당

| 장치 | 국번 |
|-----|-----|
| PLC (마스터) | - |
| 인버터 #1 (입측) | 1 |
| 인버터 #2 (압연 상) | 2 |
| 인버터 #3 (압연 하) | 3 |
| 인버터 #4 (출측) | 4 |

### PLC 통신 설정 (XGB)

- 통신 포트: COM2 (RS-485)  
- 프로토콜: Modbus RTU Master  
- 통신속도: 9600 bps  
- 패리티: None  
- 스톱비트: 1  
- 타임아웃: 200 ms  
- 폴링 주기: 50 ms

## 시운전-003 | 에너지 모니터링 시스템 (2024-07)

**현장**: A공장 에너지 모니터링  
**구성**: 오므론 전력량계 × 3 + LabVIEW PC + SQL Server

### 전력량계 사양

| 항목 | 값 |
|-----|---|
| 제조사/모델 | 오므론 KM50 |
| 입력 전압 | 3상 380V |
| 입력 전류 | CT 비율 200/5A |
| 펄스 출력 | 1000 pulse/kWh |
| 펄스 폭 | 60 ms |
| 펄스 출력 형태 | 오픈컬렉터 (12-24 Vdc) |

### PLC 고속카운터 설정 (XGB)

- 입력 채널: HSC0, HSC1, HSC2  
- 카운터 모드: 단상 32비트  
- 계산식: kWh = 누적카운터 / 1000  
- 초기화: 00:00 자정 카운터 리셋 또는 오버플로우 처리 필요

### LabVIEW 수집 설정

- Modbus TCP로 PLC 데이터 수집  
- 전력량 누적값 및 순시 전력 읽기  
- DB 저장 주기: 1분 단위 평균값  
- 일별/월별 집계는 SQL VIEW로 처리

## 시운전-004 | 로드셀 기반 중량 측정 (2024-09)

**현장**: C공장 배합 설비  
**구성**: 로드셀 × 4 + 접속박스 + PLC AI 모듈 (4-20 mA)

### Calibration 절차

1. 로드셀 배선 완료 후 무부하 출력(mV/V) 측정  
2. PLC AI 입력 count 읽기 (무부하 → Zero 기준값)  
3. 기준 분동 적재 (정격 용량의 80% 이상 권장)  
4. 분동 적재 시 count 읽기 (Span 기준값)  
5. 스케일링 파라미터 입력  
6. 여러 점에서 직선성 확인

### 노이즈 대책

- 신호 케이블: 쉬엘드 4심 (로드셀 ↔ 접속박스 ↔ 증폭기)  
- 쉬엘드 접지: PLC 판넬측 단일 접지  
- 인버터/동력 케이블과 30 cm 이상 이격  
- AI 모듈 필터 상수: 초기 100 ms 설정 후 현장 상황 따라 조정  
- 접속박스 내 서미스터 또는 션트 저항은 로드셀 사양서 확인
