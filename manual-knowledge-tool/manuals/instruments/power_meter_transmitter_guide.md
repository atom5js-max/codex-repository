# 계측기 운용 가이드: 전력량계·압력·온도 트랜스미터

## 1. 전력량계 (Power Meter)

### 펄스 출력 (Pulse Output)

전력량계는 소비 전력에 비례해 펄스를 출력한다.

| 항목 | 내용 |
|-----|------|
| 출력 형태 | 오픈컬렉터 (NPN 또는 PNP 선택) |
| 펄스 레이트 | 1000 pulse/kWh (기종마다 다름) |
| 펄스 폭(Pulse Width) | 60 ms 이상 (DI 입력 응답 속도 이상) |
| 전원 | DC 12~24 V (외부 공급) |

**전력량 계산식**:
```
E [kWh] = 누적 펄스 수 / 펄스 레이트 [pulse/kWh]
```

### PLC 연결 시 주의사항

- 펄스 폭이 PLC DI 입력 필터 설정보다 짧으면 누락 발생
  → PLC DI 입력 필터를 펄스 폭 이하로 설정하거나 고속카운터(HSC) 사용
- 오픈컬렉터 출력: 외부 풀업 저항 또는 소스형 입력 모듈 필요
- 계측기 펄스 레이트는 반드시 데이터시트에서 확인 (100/500/1000/10000 pulse/kWh 다양)

### 통신 인터페이스 (Modbus RTU / TCP)

대부분의 전력량계는 Modbus RTU 또는 Modbus TCP 를 지원한다.

주요 레지스터 (제조사별 상이, 반드시 사양서 확인):

| 데이터 | 레지스터 타입 |
|------|------------|
| 유효 전력 (kW) | Holding Register (FC03) |
| 무효 전력 (kVAR) | Holding Register |
| 적산 전력량 (kWh) | Holding Register (32bit, 2 레지스터) |
| 전류 (A) | Holding Register |
| 전압 (V) | Holding Register |
| 역률 (PF) | Holding Register |

## 2. 압력 트랜스미터 (Pressure Transmitter)

### 출력 신호 유형

| 유형 | 범위 | 특징 |
|-----|------|------|
| 4-20 mA | 2선식/3선식 | 가장 일반적, 장거리 전송 유리 |
| 0-10 V | 3선식 | 단거리, 임피던스 주의 |
| HART | 4-20 mA 위에 디지털 중첩 | 파라미터 설정 가능 |

### 4-20 mA 배선

```
[24V 공급] ──┬── [+] 트랜스미터 [─] ── [PLC AI+]
             │                          [PLC AI-] ── [0V]
             └─────────────────────────────────────
```

2선식: 루프 파워(Loop Power) 방식 — 전원과 신호가 같은 2선

**스케일링 공식**:
```
압력 = (mA - 4) / 16 × 정격 압력 [kPa 또는 bar]
```

### 배선 단선(Wire Break) 감지

PLC AI 모듈에서 Burnout 기능을 활성화하면 단선 시 count 가 최대 또는 최소로 고정된다.
프로그램에서 이 조건을 체크해 알람을 발생시킨다.

### 영점/스팬 조정

- 영점(Zero): 대기압(0 kPa) 상태에서 4.00 mA 출력 확인
- 스팬(Span): 정격 압력 인가 후 20.00 mA 출력 확인
- 현장 Trim: HART 통신기 또는 조정 스위치로 보정

## 3. 온도 트랜스미터 (Temperature Transmitter)

### 센서 유형

| 센서 | 기호 | 범위 | 특징 |
|-----|-----|------|------|
| PT100 | RTD | -200~+850°C | 정확도 높음 |
| PT1000 | RTD | -200~+850°C | PT100 고저항 버전 |
| K 타입 | TC | -200~+1370°C | 고온용 일반적 |
| J 타입 | TC | -40~+750°C | 저비용 |

### 트랜스미터 배선 (4-20 mA 출력)

RTD 3선식 배선 시 보상 도선 저항 오차 주의.
트랜스미터가 없는 경우 PLC 전용 TC/RTD 모듈 직결 가능.

### 온도 Calibration

1. 기준 온도계와 비교 측정
2. 영점(냉접점 보상) 확인
3. 스팬 조정 (고온 기준점 적용)

## 4. 공통: 아날로그 신호 노이즈 대책

- 쉴드 케이블 사용 → 한쪽(PLC 판넬)만 접지
- 인버터 출력 케이블과 30 cm 이상 이격
- 덕트 내 동력/신호 케이블 분리 배선
- PLC AI 모듈 디지털 필터 설정 (50~200 ms)
- 심한 노이즈 환경: 격리형(Isolated) AI 모듈 사용 검토
