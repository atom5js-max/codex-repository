# 로드셀 / 장력센서 증폭기 설정 가이드

## 1. 로드셀 기본 구조

로드셀은 하중에 비례한 mV/V 신호를 출력한다.

```
정격 출력 = 2 mV/V (예)
공급전압 = 5V
만부하 시 출력 = 2 × 5 = 10 mV
```

증폭기는 이 mV 신호를 4-20 mA 또는 0-10 V 로 변환한다.

---

## 2. 증폭기 출력 확인 방법

### 무부하 출력 (Zero)

- 로드셀에 아무것도 올리지 않은 상태에서 증폭기 출력 측정
- 정상: 4.00 mA ±0.1 mA (4-20 mA 출력 기준)
- 무부하에서 4 mA 를 벗어나면 Zero 조정 또는 배선 확인

### 만부하 출력 (Span)

- 정격 분동을 올린 상태에서 증폭기 출력 측정
- 정상: 20.00 mA ±0.2 mA
- 실제값과 차이나면 Calibration (Span 조정) 필요

---

## 3. 노이즈 (Noise) 대책

### 원인

- 인버터 PWM 노이즈가 신호 케이블에 유도됨
- Ground Loop: 쉴드 케이블 양쪽 접지 시 순환 전류 발생
- 기계 진동이 로드셀에 전달

### 대책

**Shield (쉴드) 접지**:
- 쉴드는 한쪽(PLC 판넬 측)만 접지
- 반대쪽(로드셀/증폭기 측)은 절연 처리
- Ground Loop 형성을 방지하기 위해

**Filter 상수**:
- 증폭기 내장 Low-Pass Filter 상수를 늘려 고주파 노이즈 제거
- 일반적으로 50~200 ms 설정 (응답속도와 안정성 트레이드오프)

**케이블 배선**:
- 동력 케이블(인버터 출력)과 최소 30 cm 이격
- 교차 시 직각 교차

**Analog Input 필터**:
- PLC AI 모듈의 디지털 필터 상수 설정
- XGB 기준: 채널별 필터 시정수 설정 가능

---

## 4. Calibration (교정) 절차

1. 설비 안정화 (무부하 상태로 5분 이상 대기)
2. 로드셀 무부하 출력(mV/V) 측정 및 기록
3. PLC AI count 읽기 (이것이 Zero 기준값)
4. 기준 분동 적재 (정격의 80% 이상 권장)
5. 분동 적재 시 AI count 읽기 (이것이 Span 기준값)
6. 스케일링 파라미터 입력

**스케일링 공식**:
```
공학값(kgf) = (count - zero_count) / (span_count - zero_count) × 정격용량
```

---

## 5. Offset (오프셋) 조정

설치 후 구조물 자중이나 편심으로 인해 무부하에서 영점이 맞지 않을 수 있다.
증폭기의 Zero 조정 포텐셔미터 또는 소프트웨어 Offset 설정으로 보정한다.

PLC 프로그램에서 소프트 오프셋 처리:
```
실제값 = 측정값 - 오프셋
```

## 6. 배선 점검 체크리스트

- [ ] 로드셀 케이블 색상 확인 (EX+/EX-/SIG+/SIG-)
- [ ] 쉴드 한쪽 접지 여부
- [ ] 증폭기 공급전압 확인 (5V 또는 10V, 사양서 확인)
- [ ] 증폭기 출력 형태 (4-20 mA / 0-10 V)
- [ ] PLC AI 모듈 입력 범위 설정 일치 여부
- [ ] 케이블 단선 여부 (테스터기로 저항 측정)
- [ ] 동력 케이블과의 이격 거리
